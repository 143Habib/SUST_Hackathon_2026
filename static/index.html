Investigation Workflow

The engine is deterministic for speed, repeatability, safety, and hidden-test stability. It does not need an external LLM for core decisions.

1

Normalize

Handles English, Bangla, Banglish, Bangla digits, amounts, transaction IDs, phone numbers, and time clues.

2

Classify

Maps the complaint to the exact allowed case taxonomy without enum drift.

3

Match Evidence

Scores transaction history by amount, type, status, counterparty, timestamp, and explicit transaction ID.

4

Route & Escalate

Selects severity, department, and human-review flag for disputes, high value, suspicious, or ambiguous cases.

5

Safe Reply

Uses guarded templates that never request PIN/OTP/password and never promise unauthorized refunds or reversals.
